## 推荐使用顺序

### 第一次使用时

按这个顺序：

1. 先看 `学习设计文档`
2. 再看 `三个月实施文档`
3. 然后看当前周对应的 `每周执行清单`
4. 最后每天只打开 `每日执行卡片`

---

## 你平时应该怎么用

### 每天

只需要打开：

- `每日执行卡片`

### 每周开始时

打开：

- `每周执行清单`
- 如果忘了当前 Sprint 目标，再看 `三个月实施文档`

### 每两周复盘时
